About the game:

Fantastic Fist is a platforming game focused on simultaneous keyboard and mouse controls. While the character can be moved around with the keyboard, the mouse can be used to interact with the environment by punching with a giant fist.

This game was solo developed by Chris Siebert. The Art, Music, Levels, Code, and everything else was all made by one person. That said, the game would never be where it is today without the feedback of the playtesters!

Featuring:

  • Hidden collectibles

  • Challenging bonus levels

  • Boxes with faces and sometimes hats
